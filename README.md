# Car Rental Management Desktop & Web Application

A dual-platform system (desktop + web) designed to manage car rental operations efficiently, ensuring real-time synchronization between bookings and vehicle availability.

---

## 📌 Overview

This project is a car rental management system consisting of two versions: a **JavaFX desktop application** and a **Symfony web application**.

It allows rental agencies to manage cars, handle bookings based on availability, and synchronize data across both platforms using a centralized MySQL database. The system also integrates SMS notifications to confirm bookings automatically.

---

## ✨ Key Features

- 🚗 Car management and availability tracking  
- 📅 Booking system with conflict prevention  
- 🔄 Real-time synchronization between desktop and web applications   
- 📲 SMS notifications for booking confirmations (Twilio)
